# Enterprise-Network-Lab
This project is a large-scale CCNA lab designed to simulate a real enterprise network environment.

The lab includes:
IPv4 and IPv6, 
static routes, 
VLANs, 
spanning tree, 
OSPF, 
EtherChannel, 
DHCP, 
DNS, 
NAT, 
SSH, 
security features, 
wireless,
and more...
